stableRoomateMatching
=====================

R implementation of the pseudo code by Stephan Mertens (2014) based on Irvin's algorithm (1985).


Example from Wikipedia 
http://en.wikipedia.org/wiki/Stable_roommates_problem

rank = matrix(
  c(
  3,4,2,6,5,  
  6,5,4,1,3,  
  2,4,5,1,6,   
  5,2,3,6,1, 
  3,1,2,4,6,
  5,1,3,4,2), 
  nrow=3);
